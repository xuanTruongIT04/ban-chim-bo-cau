3. Add a private helper method to the class:
```php
private function primaryImage(ProductModel $product): ?array
{
    if (! $product->relationLoaded('images')) {
        return null;
    }

    $primary = $product->images->firstWhere('is_primary', true);

    if ($primary === null) {
        return null;
    }

    return [
        'url'           => Storage::disk('s3')->url($primary->path),
        'thumbnail_url' => Storage::disk('s3')->url($primary->thumbnail_path),
    ];
}
```
4. Add `use App\Infrastructure\Persistence\Eloquent\Models\ProductModel;` import (already imported — the @mixin uses CartItemModel, but ProductModel type hint needs explicit import. Check existing imports first; the file already imports CartItemModel).

**Note:** Use a private helper method instead of inlining the logic in `toArray()` because `$this` refers to CartItemModel (via @mixin), not ProductModel — the `whenLoaded` pattern used in ProductResource won't work here since we're checking images on `$product` (a related model), not on `$this`. The `relationLoaded` check serves the same purpose — graceful null when images aren't eager loaded.
