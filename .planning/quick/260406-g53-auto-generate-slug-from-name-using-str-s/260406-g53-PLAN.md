2. In UpdateCategoryRequest::rules(), change slug rule from:
   `['required', 'string', 'max:255', 'unique:categories,slug,' . $this->route('category'), 'regex:/^[a-z0-9]+(?:-[a-z0-9]+)*$/']`
   to:
   `['nullable', 'string', 'max:255', 'unique:categories,slug,' . $this->route('category'), 'regex:/^[a-z0-9]+(?:-[a-z0-9]+)*$/']`
   (Only change: `required` -> `nullable`)

3. In CategoryController, add `use Illuminate\Support\Str;` import at the top.

4. In CategoryController::store(), after `$validated = $request->validated();`, add slug resolution:
   ```php
   $slug = $validated['slug'] ?? Str::slug($validated['name']);
   ```
   Then pass `$slug` instead of `$validated['slug']` to `$this->createAction->handle(slug: $slug, ...)`.

   Then pass `$slug` instead of `$validated['slug']` to `$this->updateAction->handle(slug: $slug, ...)`.
